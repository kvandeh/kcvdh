Data3

 SQLite3 wrapper for Python With a focus on being object oriented, Data3 is a wrapper for SQLite3 that simplifies database interactions. Making everything iterable and enabling you to use the same syntax as you would with a normal list. Data3 is designed to be easy to use and understand, making it a great choice for developers of all skill levels.
